# SimpleVoice
Путь `permission.yml > integration.simplevoice`

## Пояснение
Категория, отвечающая за настройку прав у интеграции SimpleVoice

## Редактирование
```yaml
<permission.integration.simplevoice>
```

### По умолчанию
```yaml
simplevoice:
  name: "flectonepulse.module.integration.simplevoice"
  type: TRUE
```
