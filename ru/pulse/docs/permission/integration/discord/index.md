# Дискорд
Путь `permission.yml > integration.discord`

## Пояснение
Категория, отвечающая за настройку прав у интеграции Discord

## Редактирование
```yaml
<permission.integration.discord>
```

### По умолчанию
```yaml
discord:
  name: "flectonepulse.module.integration.discord"
  type: TRUE
```
