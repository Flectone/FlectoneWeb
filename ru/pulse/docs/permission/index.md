# Файл `permission.yml`

## Пояснение
Файл, который содержит в себе ВСЕ права на то или иное действие

## Пояснение
Файл по настройке сообщений `FlectonePulse` (из-за чего появляются и как)

## Редактирование
```yaml
<permission>
```

### По умолчанию
```yaml
module:
  name: "flectonepulse.module"
  type: TRUE
```
