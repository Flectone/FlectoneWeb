# Значения
Путь `permission.yml > message.objective.*`

## Пояснение
Категория, отвечающая за настройку прав у значений игрока

## Редактирование
```yaml
<permission.message.objective>
```

### По умолчанию
```yaml
objective:
  name: "flectonepulse.module.message.objective"
  type: TRUE
```
