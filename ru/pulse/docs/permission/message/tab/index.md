# Таб
Путь `permission.yml > message.tab`

## Пояснение
Категория, отвечающая за настройку прав у Таба сервера

## Редактирование
```yaml
<permission.message.tab>
```

### По умолчанию
```yaml
tab:
  name: "flectonepulse.module.message.tab"
  type: TRUE
```
