# Верхнее
Путь `permission.yml > message.tab.header`

## Пояснение
Категория, отвечающая за настройку прав у верхней части Таба сервера

## Редактирование
```yaml
<permission.message.tab.header>
```

### По умолчанию
```yaml
header:
  name: "flectonepulse.module.message.tab.header"
  type: TRUE
```
