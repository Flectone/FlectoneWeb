# Сообщения
Путь `permission.yml > message.*`

## Пояснение
Категория, отвечающая за настройку прав у сообщений

## Редактирование
```yaml
<permission.message>
```

### По умолчанию
```yaml
integration:
  name: "flectonepulse.module.message"
  type: TRUE
```
