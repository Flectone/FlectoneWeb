# Спойлер
Путь `permission.yml > message.format.spoiler`

## Пояснение
Категория, отвечающая за настройку прав у тегов спойлеров

## Редактирование
```yaml
<permission.message.format.spoiler>
```

### По умолчанию
```yaml
spoiler:
  name: "flectonepulse.module.message.format.spoiler"
  type: TRUE
```
