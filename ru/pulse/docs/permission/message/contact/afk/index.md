# Афк
Путь `permission.yml > player.afk`

## Пояснение
Категория, отвечающая за настройку прав у игрока Афк

## Редактирование
```yaml
<permission.message.contact.afk>
```

### По умолчанию
```yaml
afk:
  name: "flectonepulse.module.message.contact.afk"
  type: TRUE
```
