# Версия
Путь `permission.yml > message.status.version`

## Пояснение
Категория, отвечающая за настройку прав у версии сервера

## Редактирование
```yaml
<permission.message.status.version>
```

### По умолчанию
```yaml
version:
  name: "flectonepulse.module.message.status.version"
  type: TRUE
```
