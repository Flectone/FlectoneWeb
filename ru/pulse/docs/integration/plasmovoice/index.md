# PlasmoVoice
Путь `integration.yml >plasmovoice`

## Пояснение
Интеграция с PlasmoVoice позволяет мутить игроков, которые были замучены с помощью `/mute`
![mute](/mute.png)

## Редактирование
```yaml
<integration.plasmovoice>
```

### По умолчанию
```yaml
plasmovoice:
  enable: true
```
