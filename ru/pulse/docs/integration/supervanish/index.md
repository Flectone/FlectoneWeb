# SuperVanish
Путь `integration.yml >supervanish`

## Пояснение
Интеграция с SuperVanish (и PremiumVanish) скрывает действия игроков в ванише

## Редактирование
```yaml
<integration.supervanish>
```

### По умолчанию
```yaml
supervanish:
  enable: true
```
