# Книга
Путь `message.yml > book`

## Пояснение
Модуль, отвечающий за сообщения в книге и названии этой книги
![book](/book.gif)

## Редактирование
```yaml
<message.book>
```

### По умолчанию
```yaml
book:
  enable: true
```
