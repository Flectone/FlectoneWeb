# Табличка
Путь `message.yml > sign`

## Пояснение
Модуль, отвечающий за сообщения на табличке
![sign](/signoak.gif)

## Редактирование
```yaml
<message.sign>
```

### По умолчанию
```yaml
sign:
  enable: true
  sound:
    enable: false
```
