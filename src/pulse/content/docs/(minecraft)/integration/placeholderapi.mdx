- Позволяет использовать любые [плейсхолдеры](https://github.com/PlaceholderAPI/PlaceholderAPI/wiki/Placeholders)

<Callout type="info" title="НАПример">
`%server_tps%`, `%server_online%` и т.д., если установлен соответствующий модуль с помощью `/papi ecloud download модуль`
</Callout>

- Добавляет свои плейсхолдеры

| Плейсхолдер                                 | Что возвращает                                                       |
|---------------------------------------------|----------------------------------------------------------------------|
| `%flectonepulse_ip%`                        | Возвращает IP-адрес игрока                                           |
| `%flectonepulse_online%`                    | Возвращает количество игроков на сервере                             |
| `%flectonepulse_ping%`                      | Возвращает пинг игрока                                               |
| `%flectonepulse_player%`                    | Возвращает обычное имя игрока                                        |
| `%flectonepulse_tps%`                       | Возвращает TPS сервера                                               |
| `%flectonepulse_mute_suffix%`               | Возвращает [суффикс мута](/pulse/docs/command/mute/) игрока                |
| `%flectonepulse_afk_durationx%`             | Возвращает время бездействия игрока в секундах               |
| `%flectonepulse_afk_durationx_formatted%`   | Возвращает отформатированное время бездействия игрока в секундах               |
| `%flectonepulse_condition_название%`        | Возвращает результат [условия](/pulse/docs/message/format/condition/)               |
| `%flectonepulse_fcolor_число%`              | Возвращает [кастомный цвет](/pulse/docs/message/format/fcolor/) игрока     |
| `%flectonepulse_fcolor_out_число%`          | Возвращает [кастомный цвет OUT](/pulse/docs/message/format/fcolor/) игрока |
| `%flectonepulse_fcolor_see_число%`          | Возвращает [кастомный цвет SEE](/pulse/docs/message/format/fcolor/) игрока |
| `%flectonepulse_setting_название%`          | Возвращает значение конкретной настройки у игрока <br/><br/>Если `setting` является [SettingText](https://github.com/Flectone/FlectonePulse/blob/master/core/src/main/java/net/flectone/pulse/util/constant/SettingText.java), то вернётся его значение. <br/>Иначе проверяется [ModuleName](https://github.com/Flectone/FlectonePulse/blob/master/core/src/main/java/net/flectone/pulse/util/constant/ModuleName.java), если включено то `yes`, иначе `no` <br/><br/>Например, `%flectonepulse_setting_afk_suffix%` вернёт АФК суффикс, а `%flectonepulse_setting_module_message_chat%` вернёт `yes`, если у игрока включен чат, и `no` если нет             |
