- Allows using any [placeholders](https://github.com/PlaceholderAPI/PlaceholderAPI/wiki/Placeholders)

<Callout type="info" title="Example">
    `%server_tps%`, `%server_online%` etc., if the corresponding module is installed using `/papi ecloud download module`
</Callout>

- Adds its own placeholders

| Placeholder                                 | Returns                                                       |
|---------------------------------------------|----------------------------------------------------------------------|
| `%flectonepulse_ip%`                        | Returns the player's IP address                                           |
| `%flectonepulse_online%`                    | Returns the number of players on the server                             |
| `%flectonepulse_ping%`                      | Returns the player's ping                                               |
| `%flectonepulse_player%`                    | Returns the player's regular name                                        |
| `%flectonepulse_tps%`                       | Returns the server TPS                                               |
| `%flectonepulse_mute_suffix%`               | Returns the player's [mute suffix](/pulse/docs/command/mute/)                |
| `%flectonepulse_afk_durationx%`             | Returns the player's afk duration in seconds               |
| `%flectonepulse_afk_durationx_formatted%`   | Returns the formatted player's afk duration in seconds               |
| `%flectonepulse_condition_name%`            | Returns the result of the [condition](/pulse/docs/message/format/condition/)               |
| `%flectonepulse_fcolor_number%`             | Returns the player's [custom color](/pulse/docs/message/format/fcolor/)     |
| `%flectonepulse_fcolor_out_number%`         | Returns the player's [custom color OUT](/pulse/docs/message/format/fcolor/) |
| `%flectonepulse_fcolor_see_number%`         | Returns the player's [custom color SEE](/pulse/docs/message/format/fcolor/) |
| `%flectonepulse_setting_name%`              | Returns the value of a specific setting for the player <br/><br/>If `setting` is a [SettingText](https://github.com/Flectone/FlectonePulse/blob/master/core/src/main/java/net/flectone/pulse/util/constant/SettingText.java), its value is returned. <br/>Otherwise, it checks [ModuleName](https://github.com/Flectone/FlectonePulse/blob/master/core/src/main/java/net/flectone/pulse/util/constant/ModuleName.java); if enabled, returns `yes`, otherwise `no` <br/><br/>For example, `%flectonepulse_setting_afk_suffix%` returns the AFK suffix, and `%flectonepulse_setting_module_message_chat%` returns `yes` if the player has chat enabled, and `no` if not |
