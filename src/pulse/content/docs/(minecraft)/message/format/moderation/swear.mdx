### `symbol`

Символ, которым заменяется каждая буква ругательства

### `format_see`

Формат ругательства для игрока с пермишеном [просмотра ругательств](#see) (по умолчанию только операторы сервера), обычные игроки будут видеть сообщение из символов

### `format_restrict`

Сообщение, если игрок слишком часто ругался

### `violation_limit`

Сколько раз подряд игрок должен ругаться, чтобы получить ограничение на чат

### `violation_reset_time`

Сколько времени будет действительно каждое нарушение в тиках (20 тиков = 1 секунда)

### `ignore`

Список слов, которые не являются ругательством и их нужно пропускать при проверке

### `trigger`

Список [регулярных выражений](https://javarush.com/groups/posts/regulyarnye-vyrazheniya-v-java) для того, чтобы определять ругательство

<Callout type="info" title="Например">
- Для цифренного айпи `\\b((?:\\d{1,3}\\.){3}\\d{1,3})\\b`
- Для буквенного айпи `(?i)\\b((https?:\\/\\/)?([\\w\\.а-яА-Я-]+\\.([a-z]{2,4}|[рР][фФ]|[уУ][кК][рР]))\\b(:\\d{2,7})?(\\/\\S+)?)`
</Callout>
