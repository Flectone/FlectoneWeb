![command whois](/pulse/command_whois.webp)

По умолчанию, каждая цветная кнопка используется, чтобы вызвать соответствующую комманду: `/online total`, `/geolocate`, `/whitelistplayer list`, `/warnlist`, `/mutelist`, `/banlist`


Комманду можно использовать с айпи, а также с игроком. Если введён игрок, то будет использоваться его последний айпи

### `empty`

Сообщение, если игроков с введённым айпи не найдено

### `null_page`

Сообщение, если введённая страница не существует

### `online`

Формат для `<online>`, если игрок с таким айпи находится в сети

### `offline`

Формат для `<online>`, если игрок с таким айпи не в сети

### `header`

Верхняя часть сообщения списка

### `line`

Формат каждого предупреждения из списка

### `footer`

Нижняя часть сообщения списка
