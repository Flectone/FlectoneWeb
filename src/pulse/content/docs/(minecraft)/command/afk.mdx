### `suffix`

Суффикс, который будет отображатся рядом с ником игрока

### `format_true`

<Accordions>
    <Accordion title="Настройка формата сообщения, если игрок отошел">
        #### `global`

        Сообщение, которое увидять все игроки

        #### `local`

        Сообщение, которое увидит только игрок, который отошел
    </Accordion>
</Accordions>

### `format_false`

<Accordions>
    <Accordion title="Настройка формата сообщения, если игрок вернулся">
        #### `global`

        Сообщение, которое увидять все игроки

        #### `local`

        Сообщение, которое увидит только игрок, который вернулся
    </Accordion>
</Accordions>
