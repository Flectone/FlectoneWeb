![command online](/pulse/commandonline.png)

In any message, you can use the `<online:type>` placeholder to display the number of seconds of a specific type (`first`, `last`, `total`). To show formatted time, use `<online:type_formatted>`. For example, `<online:total_formatted>` returns the formatted total playtime on the server.

### `null_player`

Message if the entered player is not found

### `format_current`

Message if the entered player is on the server

### `format_first`

Format of the `/online first` message that will be sent

### `format_last`

Format of the `/online last` message that will be sent

### `format_total`

Format of the `/online total` message that will be sent
