Integration with the `TAB` plugin resolves compatibility issues by disabling certain modules in `FlectonePulse`:
- [Scoreboard](/pulse/docs/message/format/scoreboard/)
- [Header](/pulse/docs/message/tab/header/)
- [Footer](/pulse/docs/message/tab/footer/)
- [Player List Name](/pulse/docs/message/tab/playerlistname/)

### `disable_flectonepulse_scorebord`

Disables the [Scoreboard](/pulse/docs/message/format/scoreboard/) module

### `disable_flectonepulse_header`

Disables the [Header](/pulse/docs/message/tab/header/) module

### `disable_flectonepulse_footer`

Disables the [Footer](/pulse/docs/message/tab/footer/) module

### `disable_flectonepulse_playerlistname`

Disables the [Player List Name](/pulse/docs/message/tab/playerlistname/) module
