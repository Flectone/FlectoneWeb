<Callout type="warn" title="Внимательно">
Это интеграция с другим плагином на `TAB`, настроить сам `TAB` можно [тут](/pulse/docs/message/tab/)
</Callout>

Интеграция с плагином `TAB` устраняет проблемы совместимости, выключая работу некоторых модулей в `FlectonePulse`:
- [Скорборд](/pulse/docs/message/format/scoreboard/)
- [Верхнее](/pulse/docs/message/tab/header/)
- [Нижнее](/pulse/docs/message/tab/footer/)
- [Имя в списке](/pulse/docs/message/tab/playerlistname/)
