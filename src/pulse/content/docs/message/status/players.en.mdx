### `samples`

List of server players. The `<players>` placeholder adds all server players to the list.

<Callout type="info" title="USAGE EXAMPLES">
    You can add any messages
    ```yaml
    samples:
    - name: "<red>This is a Minecraft server!"
    id: null
    - name: "<players>"
        id: null
        ```
        ![sample1](/pulse/sample1.png)

        or create fake players
        ```yaml
        samples:
        - name: "Notch"
        id: null
        - name: "<players>"
            id: null
            ```
            ![sample2](/pulse/sample2.png)
</Callout>

<Callout type="warn" title="Warning">
    Only available colors can be used
    <include>../../../../parts/color.en.mdx</include>
</Callout>

### `control`

If enabled, the player will not be allowed to join the server if there are more players than [specified](#max)

### `max`

Maximum number of players on the server

### `online`

A number that allows you to specify any online count on the server

<Callout type="info" title="Information">
    The number can be anything, but `-69` indicates the current online count
</Callout>
