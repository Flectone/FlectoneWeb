### `ignore`

List of words that are not profanity and should be skipped swear checking

### `trigger`

A list of [regular expressions](https://en.wikipedia.org/wiki/Regular_expression) used to identify profanity

<Callout type="info" title="For example">
    - For a numeric IP: `\\b((?:\\d{1,3}\\.){3}\\d{1,3})\\b`
    - For a domain/URL: `(?i)\\b((https?:\\/\\/)?([\\w\\.а-яА-Я-]+\\.([a-z]{2,4}|[рР][фФ]|[уУ][кК][рР]))\\b(:\\d{2,7})?(\\/\\S+)?)`
</Callout>
