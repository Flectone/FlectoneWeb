### `mode`

Режим определения мира из доступных

| Режим    | Описание                                                                 |
|----------|--------------------------------------------------------------------------|
| `TYPE`   | Мир определяется по названию его типа, например обычный мир это `normal` |
| `NAME`   | Мир определяется по названию его папки                                   |
