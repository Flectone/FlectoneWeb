### `mode`

The mode for determining the world from those available

| Mode     | Description                                                                     |
|----------|---------------------------------------------------------------------------------|
| `TYPE`   | The world is determined by its type name; for example, a standard world is `normal` |
| `NAME`   | The world is determined by its folder name                                      |
