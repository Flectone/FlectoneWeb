### `cooldown`

Message displayed when a feature is used too frequently, if `cooldown` is enabled

### `time`

Time-related messages

- #### `format`

Time formatting ([Apache DurationFormatUtils](https://commons.apache.org/proper/commons-lang/apidocs/org/apache/commons/lang3/time/DurationFormatUtils.html))

- #### `permanent`

Time formatting if the value is `-1`, i.e., permanent

- #### `zero`

Time formatting if the value is zero
