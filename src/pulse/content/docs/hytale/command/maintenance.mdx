### `already_true`

Сообщение, если технические работы уже включены

### `already_false`

Сообщение, если технические работы уже выключены

### `server_description`

Сообщения при просмотре сервера, если включены технические работы

### `server_version`

Название версии сервера, если включены технические работы

### `reasons`

Список с ключами и значениями, где ключом является слово, а значением конкретная причина

<Callout type="info" title="Можно вписывать свои причины, например">
    ```yaml
    random_kek: "Random reason"
    ```
    Тогда если я напишу `/maintenance start 1d random_kek`, то причиной будет `Random reason`

    Если причина не указана, будет использоваться `default`
</Callout>

### `format_true`

Сообщение при включении технических работ

### `format_false`

Сообщение при отключении технических работ

### `person`

Сообщение игроку, который попытался подключиться во время технических работ
