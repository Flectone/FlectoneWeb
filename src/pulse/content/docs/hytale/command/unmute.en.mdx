### `null_player`

Message if the entered player is not found

### `null_id`

Message if the entered moderation ID is incorrect

### `not_muted`

Message if the entered player is not muted

### `lower_weight_group`

Message if the command is executed by a player with a group lower than the player being moderated

### `reasons`

A list with keys and values, where the key is a word and the value is a specific reason

<Callout type="info" title="Information">
    You can add your own reasons, for example

    ```yaml
    random_kek: "Random reason"
    ```
    Then if I write `/unmute Notch random_kek`, the reason will be `Random reason`

    If no reason is specified, `default` will be used
</Callout>

### `format`

Message upon unmuting
