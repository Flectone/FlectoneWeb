- Чтобы изменить никнейм другому игроку, нужно использовать `/nicknameother игрок`
- Чтобы сбросить никнейм, нужно вписать `clear` т.е. `/nickname clear`

### `null_player`

Сообщение, если введённый игрок не найден

### `null_nickname`

Сообщение, если введённый ник нельзя использовать

### `default_nickname`

Ник, который будет использоваться по умолчанию, если кастомный никнейм не установлен

### `display`

Как будет отображаться изменённый никнейм всем

### `display_see`

Как будет отображаться изменённый никнейм игроку с правом `see`

### `format`

Сообщение при изменении никнейма
