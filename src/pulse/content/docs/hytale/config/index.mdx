### `server`

Название сервера, которое используется для идентификации в прокси режиме и в командах модерации. Это случайное значение, которое генерируется автоматически при первом включении плагина

Можно использовать плейсхолдер `<server>` или `%flectonepulse_setting_server%` для получения названия сервера у конкретного игрока. Это можно удобно использовать в модуле `condition` для разного отображения сообщений

<Callout type="warn" title="Предупреждение">
    1. На разных серверах должно быть разное значение `server`
    2. Будь внимателен при создании чистого сервера, потому что все модерации (баны, вайтлист и т.д.) сохраняются относительно этого названия. Если ты удалил/очистил конфиг, а базу данных оставил старую, то сохрани значение `server` в config.yml обязательно
</Callout>

### `version`

Текущая версия плагина, используемая для корректного перехода между версиями плагина

<Callout type="warn" title="Предупреждение">
    Не изменяй версию плагина, если не знаешь, что делаешь!
</Callout>
