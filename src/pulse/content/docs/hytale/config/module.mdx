### `enable`

Включает или выключает работу всех модулей

1. Каждый модуль имеет родителя (тот, кто выше находится)

<Callout type="info" title="Например">
```yaml
house:
  door:
  chair:
```

`house` является родителем для `door` и `chair`
</Callout>

2. Каждый модуль зависит от `enable` родителя

<Callout type="info" title="Например">
```yaml
house:
  enable: false
  door:
    enable: true
  chair:
    enable: true
```

`house` выключен, значит внезависимости от того, включены ли `door` и `chair` - они тоже будут выключены
</Callout>
