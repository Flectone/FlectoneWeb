### `empty`

Message if the list of muted players is empty

### `null_page`

Message if the entered page does not exist

### `null_player`

Message if the entered player is not found

### `global`

<Accordions>
    <Accordion title="Messages for global mutes">
        #### `header`

        Top part of the list message

        #### `line`

        Format of each mute entry in the list

        #### `footer`

        Bottom part of the list message
    </Accordion>
</Accordions>
