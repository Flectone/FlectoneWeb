### `empty`

Сообщение, если список игнорируемых игроков пуст

### `null_page`

Сообщение, если введённая страница не существует

### `header`

Верхняя часть сообщения списка

### `line`

Формат каждого игнорирования из списка

### `footer`

Нижняя часть сообщения списка
