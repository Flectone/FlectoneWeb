### `strategy`

Strategies where the `key` is the winner over the `value`

<Callout type="info" title="For example, if I want `scissors` to beat `paper`">
    Then I should write
    ```yaml
    scissors:
    - "paper"
    ```

    There can be multiple values, i.e.
    ```yaml
    scissors:
    - "paper"
    - "newitem"
    ```

    You can create your own keys and values
    ```yaml
    customvalue:
    - "customobject"
    ```
</Callout>
