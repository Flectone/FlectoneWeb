### `null_player`

Сообщение, если введённый игрок не найден

### `lower_weight_group`

Сообщение, если комманда исполняется от игрока с группой ниже, чем у игрока на которого применяется модерация

### `reasons`

Список с ключами и значениями, где ключом является слово, а значением конкретная причина

<Callout type="info" title="Можно вписывать свои причины, например">
```yaml
random_kek: "Random reason"
```
Тогда если я напишу `/kick player random_kek`, то причиной будет `Random reason`

Если причина не указана, будет использоваться `default`

</Callout>

### `server`

Сообщение для всех

### `person`

Сообщение для игрока
