![command chatsetting](/pulse/commandchatsetting.gif)

<Callout type="info" title="NOTE">

    You can open someone else's settings using `/chatsetting TheFaser`\
    \
    If you want to create your own menu via another plugin, you can change values using `/chatsetting TheFaser ADVANCEMENT`. This will enable or disable the display of achievements depending on the state. You can find out the state via the placeholder `%flectonepulse_advancement%`.\
    \
    Some settings can be assigned values, for example `/chatsetting TheFaser STREAM_PREFIX kek` will set the player's streamer prefix to `kek`.

### `no_permission`

Message if a specific permission is required to change a setting

### `disabled_self`

Message displayed when a player has disabled the function and tries to use it

### `disabled_other`

Message displayed when a player attempts to use the function on another player who has disabled it

### `inventory`

Title for the main GUI

### `checkbox`

<Accordions>
    <Accordion title="Toggle settings">
        #### `enabled_color`

        Color if the setting is enabled. Replaces `<status_color>`

        #### `enabled_hover`

        Message if the setting is enabled

        #### `disabled_color`

        Color if the setting is disabled. Replaces `<status_color>`

        #### `disabled_hover`

        Message if the setting is disabled

        #### `types`

        List of setting types and their names
    </Accordion>
</Accordions>

### `menu`

<Accordions>
    <Accordion title="Settings for additional menus">
        #### `item`

        Name and description of the item that will be in the main menu

        #### `inventory`

        GUI title in the additional menu

        #### `types`

        List of items with their names and descriptions
    </Accordion>
</Accordions>

### `menu`

<Accordions>
    <Accordion title="Settings for additional menus">
        #### `material`

        Item that will redirect to additional menus

        #### `slot`
        <Callout type="info" title="Information">
            To disable the menu, you can use `slot: -1`
        </Callout>

        Which slot will be in the main inventory

        #### `types`

        List of setting types with their parameters
    </Accordion>
</Accordions>

| View             | Description                                                                                                                                                                                                                                |
|------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| for `CHAT`       | The `name` field is the title, and `material` is the item type that will be in the inventory for this chat                                                                                                                                 |
| for `SEE` and `OUT` | The `name` field is the title, and `material` is the item type that will be in the inventory for this chat. It also has `colors`, where the key is a number and the value is the color to be applied. Use `""` to keep the color unchanged |
