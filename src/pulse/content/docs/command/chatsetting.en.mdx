<Callout type="info" title="NOTE">

    You can open someone else's settings using `/chatsetting TheFaser`

    If you want to create your own menu via another plugin, you can change values using `/chatsetting TheFaser ADVANCEMENT`. This will enable or disable the display of achievements depending on the state. You can find out the state via the placeholder `%flectonepulse_advancement%`.

    Some settings can be assigned values, for example `/chatsetting TheFaser STREAM_PREFIX kek` will set the player's streamer prefix to `kek`.
