### `cooldown`

Usage delay

<Accordions>
    <Accordion title="How to use?">
        <Steps>
            <Step>
                Replace `false` with `true`
            </Step>
            <Step>
                Restart `FlectonePulse` using the `/flectonepulse reload` command
            </Step>
        </Steps>

        <Callout type="info" title="">
            Additional parameters for delay settings will appear automatically
        </Callout>

        ```yaml
        cooldown:
          enable: true
          duration: 60
        ```

        | Field      | Explanation                                                                                             |
        |------------|---------------------------------------------------------------------------------------------------------|
        | `enable`   | Enables the cooldown functionality                                                                      |
        | `duration` | How many [ticks](https://minecraft.wiki/w/Tick) must pass between uses                                  |
    </Accordion>
</Accordions>
