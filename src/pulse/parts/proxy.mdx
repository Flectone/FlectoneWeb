    <Accordion title="Прокси режим работы через другой IP (решает проблемы недоступности сервиса)">
        #### `type`

        Режим работы прокси

        | Тип                           | Объяснение                                        |
        |-------------------------------|---------------------------------------------------|
        | `DIRECT `                     | Все запросы идут через основной IP (без прокси)   |
        | `HTTP`                        | Все запросы идут через HTTP прокси                |
        | `SOCKS`                       | Все запросы идут через SOCKS v5 прокси            |

        #### `host`

        Айпи адрес прокси

        #### `port`

        Порт прокси

        #### `user`

        Пользователь для авторизации прокси, может быть пустым. Можно использовать environment variables, например `${VALUE}`

        #### `password`

        Пароль для авторизации прокси, может быть пустым. Можно использовать environment variables, например `${VALUE}`
