### `sound`

Проигрывание звука

<Accordions>
    <Accordion title="Как использовать?">
        <Steps>
            <Step>
                Заменить `false` на `true`
            </Step>
            <Step>
                Перезапустить `FlectonePulse` с помощью комманды `/flectonepulse reload`
            </Step>
        </Steps>

        <Callout type="info" title="">
            Дополнительные параметры для настройки задержки появятся сами
        </Callout>

        ```yaml
        sound:
        enable: true
        volume: 1.0
        pitch: 1.0
        category: "HOSTILE"
        name: "minecraft:entity.llama.spit"
        ```

        | Поле       | Объяснение                                                                                                        |
        |------------|------------------------------------------------------------------------------------------------------------------|
        | `enable`   | Включает работу звука                                                                                            |
        | `volume`   | Громкость звука от `0` до `1.0`                                                                                  |
        | `pitch`    | Тональность звука от `0` до `1.0`                                                                                |
        | `category` | Категория из `MASTER`, `MUSIC`, `RECORD`, `WEATHER`, `BLOCK`, `HOSTILE`, `NEUTRAL`, `PLAYER`, `AMBIENT`, `VOICE` |
        | `name`     | Название звука, например `minecraft:entity.llama.spit`                                                           |
    </Accordion>
</Accordions>
