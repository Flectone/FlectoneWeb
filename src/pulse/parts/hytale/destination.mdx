### `destination`

Куда будет отправлено сообщение

| Тип           | Объяснение                                    | Требует дополнительной настройки? |
|---------------|----------------------------------------------|-----------------------------------|
| `CHAT`        | Сообщение в чат                              | Нет                               |
| `TITLE`       | Сообщение на весь экран                      | Да                                |
| `SUBTITLE`    | Сообщение на весь экран (верхняя строчка)     | Да                                |
| `TOAST`       | Сообщение в правом нижнем углу (уведомления) | Да                                |

            <Accordions>
                <Accordion title={<code>TITLE (или SUBTITLE)</code>}>
                    ![title](/pulse/hytale/destinationtitle.png)

                    ```yaml
                    destination:
                      type: "TITLE" (или "SUBTITLE")
                      subtext: ""
                      times:
                        fade_in: 20
                        stay: 60
                        fade_out: 20
                    ```

                    `subtext`

                    Сообщение для второй части отображения. Например, если выбран `TITLE`, то `subtext` будет показываться над ним и наоборот

                    `times`

                    | Поле       | Объяснение                            |
                    |------------|--------------------------------------|
                    | `fade_in`  | Время в тиках на появление сообщения |
                    | `stay`     | Время в тиках на удержание сообщения |
                    | `fade_out` | Время в тиках на удаления сообщения  |

                </Accordion>
            </Accordions>

            <Accordions>
                <Accordion title={<code>TOAST</code>}>
                    ![toast](/pulse/hytale/destinationtoast.png)
                    ```yaml
                    destination:
                      type: "TOAST"
                      icon: "Ingredient_Stick"
                      subtext: ""
                      style: "TASK"
                    ```

                    `icon`

                    Аватарка достижения, которая может быть любым предметом из Hytale. Если введён неправильный ID, то аватарки не будет

                    `subtext`

                    Описание достижения, текст снизу

                    `style`

                    Не используется
                </Accordion>
            </Accordions>
        </div>
    </Accordion>
</Accordions>
