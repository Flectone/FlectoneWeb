#### `name`

Permission name

#### `type`

Permission type

| Type     | Explanation                                                                  |
|----------|------------------------------------------------------------------------------|
| `TRUE`   | Available to any player                                                      |
| `FALSE`  | Available to no one, must be granted separately (e.g., using LuckPerms)      |
| `OP`     | Available to server operators                                                |
| `NOT_OP` | Available to NON-operators of the server                                     |
