| Mode     | Returns                              |
|----------|--------------------------------------|
| `HEALTH` | Player's current HP value            |
| `LEVEL`  | Player's current level value         |

| `PING`   | Player's current ping                |
| `ARMOR`  | Player's current armor value         |
