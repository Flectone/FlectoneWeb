### `strategy`

Стратегии, где `ключом` является тот, кто побеждает `значение`

<Callout type="info" title="Например я хочу, чтобы `ножницы` побеждали `paper`">
Значит я должен написать
```yaml
scissors:
  - "paper"
```

Значений может быть несколько т.е.
```yaml
scissors:
  - "paper"
  - "newitem"
```

Можно сделать свои ключи и значения
```yaml
customvalue:
  - "customobject"
```
</Callout>
