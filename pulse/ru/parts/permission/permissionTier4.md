#### `name`

Название права

#### `type`

Вид права

| Вид      | Объяснение                                                                     |
|----------|--------------------------------------------------------------------------------|
| `TRUE`   | Доступно для любого игрока                                                     |
| `FALSE`  | Никому не доступно, должно быть выдано отдельно (например с помощью LuckPerms) |
| `OP`     | Доступно для операторов сервера                                                |
| `NOT_OP` | Доступно для НЕ операторов сервера                                             |
