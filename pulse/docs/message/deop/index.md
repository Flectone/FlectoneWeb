# Деоп
Путь `message.yml > deop`

## Пояснение
Сообщение от сервера при использовании `/deop`
![deop](/deop.png)

## Редактирование
```yaml
<message.deop>
```

### По умолчанию
```yaml
deop:
  enable: true
  destination:
    type: CHAT
  sound:
    enable: false
```
