# SkinsRestorer
Путь `integration.yml >skinsrestorer`

## Пояснение
Интеграция с SkinsRestorer позволяет получать скины **оффлайн игроков**
![skinsrestorer](/skinsrestorer.png)

## Редактирование
```yaml
<integration.skinsrestorer>
```

### По умолчанию
```yaml
skinsrestorer:
  enable: true
```
