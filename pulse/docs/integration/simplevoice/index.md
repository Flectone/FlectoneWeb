# SimpleVoice
Путь `integration.yml >simplevoice`

## Пояснение
Интеграция с SimpleVoice позволяет мутить игроков, которые были замучены с помощью `/mute`
![mute](/mute.png)

## Редактирование
```yaml
<integration.simplevoice>
```

### По умолчанию
```yaml
simplevoice:
  enable: true
```
