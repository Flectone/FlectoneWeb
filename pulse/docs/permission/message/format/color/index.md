# Цвета
Путь `permission.yml > message.format.color`

## Пояснение
Категория, отвечающая за настройку прав у тегов цвета

## Редактирование
```yaml
<permission.message.format.color>
```

### По умолчанию
```yaml
color:
  name: "flectonepulse.module.message.format.color"
  type: TRUE
```
