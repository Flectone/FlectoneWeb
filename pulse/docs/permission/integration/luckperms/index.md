# LuckPerms
Путь `permission.yml > integration.luckperms`

## Пояснение
Категория, отвечающая за настройку прав у интеграции LuckPerms

## Редактирование
```yaml
<permission.integration.luckperms>
```

### По умолчанию
```yaml
luckperms:
  name: "flectonepulse.module.integration.luckperms"
  type: TRUE
```
