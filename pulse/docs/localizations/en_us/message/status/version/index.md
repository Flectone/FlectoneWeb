# Версия
Путь `localizations > en_us.yml > message.status.version`

## Пояснение
Название версии сервера
![version](/version.png)

## Редактирование
```yaml
<en_us.message.status.version>
```

### По умолчанию
```yaml
version:
  name: "Minecraft server"
```
