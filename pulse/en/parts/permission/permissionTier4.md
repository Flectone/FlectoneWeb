#### `name`

Permission name

#### `type`

Permission type

| Type      | Explanation                                                   |
|-----------|--------------------------------------------------------------|
| `TRUE`    | Available for any player                                     |
| `FALSE`   | Available if granted separately (e.g., via LuckPerms)        |
| `OP`      | Available for server operators                               |
| `NOT_OP`  | Available if granted separately (e.g., via LuckPerms)        |
