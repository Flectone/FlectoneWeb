# InteractiveChat
Путь `integration.yml >interactivechat`

## Пояснение
Интеграция с InteractiveChat исправляет проблемы совместимости, насколько это возможно

## Редактирование
```yaml
<integration.interactivechat>
```

### По умолчанию
```yaml
interactivechat:
  enable: true
```
