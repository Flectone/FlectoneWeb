# Статус
Путь `message.yml > status.*`

## Пояснение
Модуль, который отвечает за всё, что происходит при просмотре сервера перед подключением

## Редактирование
```yaml
<message.status>
```

### По умолчанию
```yaml
status:
  enable: true
```
