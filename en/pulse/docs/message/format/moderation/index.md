# Модерация
Путь `message.yml > format.moderation.*`

## Пояснение
Модуль, отвечающий за модерацию сообщений

## Редактирование
```yaml
<message.format.moderation>
```

### По умолчанию
```yaml
moderation:
  enable: true
```
