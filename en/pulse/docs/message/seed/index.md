# Сид
Путь `message.yml > seed`

## Пояснение
Сообщение от сервера при использовании `/seed`
![seed](/seed.png)

## Редактирование
```yaml
<message.seed>
```

### По умолчанию
```yaml
seed:
  enable: true
  destination:
    type: CHAT
  sound:
    enable: false
```
