# Наковальня
Путь `message.yml > anvil`

## Пояснение
Модуль, отвечающий за сообщения в наковальне при переименовании
![anvil](/anvil.gif)

## Редактирование
```yaml
<message.anvil>
```

### По умолчанию
```yaml
anvil:
  enable: true
```
