# Скорборд
Путь `message.yml > scoreboard`

## Пояснение
Сообщения в правой части экрана
![scoreboard](/scoreboard.png)

## Редактирование
```yaml
<message.scoreboard>
```

### По умолчанию
```yaml
scoreboard:
  enable: false
  random: true
  ticker:
    enable: true
    period: 100
```
