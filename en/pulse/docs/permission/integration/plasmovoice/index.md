# PlasmoVoice
Путь `permission.yml > integration.plasmovoice`

## Пояснение
Категория, отвечающая за настройку прав у интеграции PlasmoVoice

## Редактирование
```yaml
<permission.integration.plasmovoice>
```

### По умолчанию
```yaml
plasmovoice:
  name: "flectonepulse.module.integration.plasmovoice"
  type: TRUE
```
