# Vault
Путь `permission.yml > integration.vault`

## Пояснение
Категория, отвечающая за настройку прав у интеграции Vault

## Редактирование
```yaml
<permission.integration.vault>
```

### По умолчанию
```yaml
vault:
  name: "flectonepulse.module.integration.vault"
  type: TRUE
```
