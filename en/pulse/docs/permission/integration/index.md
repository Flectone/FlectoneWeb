# Интеграции
Путь `permission.yml > integration.*`

## Пояснение
Категория, отвечающая за настройку прав у интеграций

## Редактирование
```yaml
<permission.integration>
```

### По умолчанию
```yaml
integration:
  name: "flectonepulse.module.integration"
  type: TRUE
```
