# Телеграм
Путь `permission.yml > integration.telegram`

## Пояснение
Категория, отвечающая за настройку прав у интеграции Telegram

## Редактирование
```yaml
<permission.integration.telegram>
```

### По умолчанию
```yaml
telegram:
  name: "flectonepulse.module.integration.telegram"
  type: TRUE
```
