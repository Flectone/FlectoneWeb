# Оп
Путь `permission.yml > message.op`

## Пояснение
Категория, отвечающая за настройку прав у сообщений `/op` сервера

## Редактирование
```yaml
<permission.message.op>
```

### По умолчанию
```yaml
op:
  name: "flectonepulse.module.message.op"
  type: TRUE
  sound:
    name: "flectonepulse.module.message.op.sound"
    type: TRUE
```
