# Книга
Путь `permission.yml > message.book`

## Пояснение
Категория, отвечающая за настройку прав у сообщений игрока в книге

## Редактирование
```yaml
<permission.message.book>
```

### По умолчанию
```yaml
book:
  name: "flectonepulse.module.message.book"
  type: TRUE
```
