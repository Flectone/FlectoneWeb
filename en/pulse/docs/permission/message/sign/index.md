# Табличка
Путь `permission.yml > message.sign`

## Пояснение
Категория, отвечающая за настройку прав у сообщений игрока в табличке

## Редактирование
```yaml
<permission.message.sign>
```

### По умолчанию
```yaml
sign:
  name: "flectonepulse.module.message.sign"
  type: TRUE
```
