## Редактирование
```yaml
<permission.message.contact.sign>
```

```yaml
sign:

  type: TRUE
  cooldown-bypass:
    name: "flectonepulse.module.message.contact.sign.cooldown.bypass"
    type: OP

    name: "flectonepulse.module.message.contact.sign.sound"
    type: TRUE
```
