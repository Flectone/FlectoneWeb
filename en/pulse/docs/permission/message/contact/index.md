# Контактирование
Путь `permission.yml > message.contact.*`

## Пояснение
Категория, отвечающая за настройку прав у взаимодействий

## Редактирование
```yaml
<permission.message.contact>
```

### По умолчанию
```yaml
contact:
  name: "flectonepulse.module.message.contact"
  type: TRUE
```
